تطبيق ويب متكامل لإدارة المهام اليومية، يدعم اللغة العربية والإنجليزية، ويعمل كتطبيق جوال (PWA)، مع نظام حسابات يضمن خصوصية كل مستخدم.

---

## ✨ المميزات

- ✅ إنشاء، تعديل، حذف المهام
- ✅ بحث وتصفية المهام (الكل / مكتملة / قيد الانتظار)
- ✅ ترتيب المهام بالسحب والإفلات
- ✅ إحصائيات ونسبة إنجاز
- ✅ دعم كامل للغة العربية (RTL) والإنجليزية (LTR)
- ✅ نظام تسجيل دخول وحسابات خاصة
- ✅ تطبيق ويب تقدمي (PWA) قابل للتثبيت على الهواتف
- ✅ تصميم متجاوب مع جميع الأجهزة

## 🛠️ التقنيات المستخدمة

- **الخلفية**: Django 5.0.6
- **قاعدة البيانات**: SQLite (سهل التبديل)
- **الواجهة**: Bootstrap 5 + RTL
- **السكريبتات**: JavaScript (Dragula، Fetch API)
- **PWA**: Service Worker + Manifest
